# Docker1
Primera práctica con docker de Diseño de Interfaces
El proyecto consiste en una aplicación web desarrollada con FastAPI y Jinja2 que muestra un informe de incidencias, permitiendo aplicar filtros por categoría y gravedad mínima, además de visualizar gráficos.

# Tecnologías utilizadas
- Python 3
- FastAPI
- Jinja2
- Docker
- Docker Compose
- Chart.js

  # Funcionalidad
- Listado de incidencias
- Filtros por categoría y gravedad
- Resumen con totales y porcentajes
- Gráficos de incidencias por categoría y gravedad

# Ejecución del proyecto Ejemplo
Con Docker y Docker Compose instalados:

```bash
docker-compose up --build
